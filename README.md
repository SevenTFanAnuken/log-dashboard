## What is log-dashboard?

log-dashboard is a tool designed to help you watch over your system logs. It collects information from Apache servers, SSH services, and Windows Event logs. Then it looks for unusual activities, like brute force attacks, people trying to access hidden folders, or logins during odd hours.

This dashboard shows you these findings in a clear way. You do not need to have technical experience or know how to handle logs. It works on your Windows computer and uses familiar interfaces you can use with a mouse and keyboard.

---

## 💻 System Requirements

- Windows 10 or newer
- At least 4 GB of RAM
- 500 MB of free disk space
- Internet connection for first setup
- Web browser (Chrome, Firefox, Edge, or similar)

This software uses React and Node.js behind the scenes. You do not need to install these yourself—the program includes what it needs.

---

## 🔍 Features

- Shows security alerts from Apache web server logs.
- Monitors SSH logins to spot unauthorized access attempts.
- Checks Windows Event logs for off-hours activities.
- Detects brute force attacks automatically.
- Flags directory browsing attempts.
- Displays alerts in a simple, easy to use dashboard.
- Updates data in real-time with no user setup.
- Runs on your local machine without sending data to external servers.

## 🚀 Running log-dashboard for the First Time

### Step 1: Start the application

After installing, launch log-dashboard from the Start menu or desktop icon. 

### Step 2: Connect to your logs

The program will guide you to connect to Apache, SSH, and Windows Event logs on your system. For Windows Event logs, it uses built-in permissions to access the data. For Apache and SSH, it reads log files stored on your machine or local network.

If you do not have Apache or SSH logs, the dashboard will just show information for the logs you do have.

### Step 3: Explore the dashboard

You will see alerts arranged by security type. Each alert shows details like date, time, and IP addresses involved. You can click on alerts to learn more.
